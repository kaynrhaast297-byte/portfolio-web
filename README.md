# portfolio-web

Portfólio estático simples em HTML, CSS e JavaScript.

Estrutura: `index.html`, `style.css`, `script.js`, `assets/`.
Uso local: abra `index.html` no navegador ou use um servidor estático.

Como executar:

```bash
# abrir diretamente no navegador
start index.html

# ou usar um servidor simples com npm (opcional)
# instale um servidor como 'serve' ou 'http-server'
npm i -g serve
serve .
```

Melhorias aplicadas:
- meta tags para SEO e theme-color
- formulário com labels e feedback acessível
- botão "Ver Projetos" faz scroll suave para a seção

Contribuições:
- posso adicionar deploy automatizado (Netlify/Vercel) se desejar.
